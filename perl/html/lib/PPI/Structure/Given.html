NAME

PPI::Structure::Given - Circular braces for a switch statement

SYNOPSIS

  given ( something ) {
      ...
  }

INHERITANCE

  PPI::Structure::Given
  isa PPI::Structure
      isa PPI::Node
          isa PPI::Element

DESCRIPTION

PPI::Structure::Given is the class used for circular braces that contain the thing to be matched in a switch statement.

METHODS

PPI::Structure::Given has no methods beyond those provided by the standard the PPI::Structure manpage, the PPI::Node manpage and the PPI::Element manpage methods.

Got any ideas for methods? Submit a report to rt.cpan.org!
