NAME

PPI::Structure::Condition - Round braces for boolean context conditions

SYNOPSIS

  if ( condition ) {
      ...
  }
  
  while ( condition ) {
      ...
  }

INHERITANCE

  PPI::Structure::Condition
  isa PPI::Structure
      isa PPI::Node
          isa PPI::Element

DESCRIPTION

PPI::Structure::Condition is the class used for all round braces that represent boolean contexts used in various conditions.

METHODS

PPI::Structure::Condition has no methods beyond those provided by the standard the PPI::Structure manpage, the PPI::Node manpage and the PPI::Element manpage methods.

Got any ideas for methods? Submit a report to rt.cpan.org!
