NAME

PPI::Token::Quote::Interpolate - The interpolation quote-like operator

INHERITANCE

  PPI::Token::Quote::Interpolate
  isa PPI::Token::Quote
      isa PPI::Token
          isa PPI::Element

DESCRIPTION

A PPI::Token::Quote::Interpolate object represents a single interpolation quote-like operator, such as qq{$foo bar $baz}.

METHODS

There are no methods available for PPI::Token::Quote::Interpolate beyond those provided by the parent the PPI::Token::Quote manpage, the PPI::Token manpage and the PPI::Element manpage classes.

Got any ideas for methods? Submit a report to rt.cpan.org!
