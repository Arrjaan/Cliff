NAME

Moose::Meta::TypeConstraint::Parameterizable - Type constraints which can take a parameter (ArrayRef)

VERSION

version 2.0205

DESCRIPTION

This class represents a parameterizable type constraint. This is a type constraint like ArrayRef or HashRef, that can be parameterized and made more specific by specifying a contained type. For example, instead of just an ArrayRef of anything, you can specify that is an ArrayRef[Int].

A parameterizable constraint should not be used as an attribute type constraint. Instead, when parameterized it creates a the Moose::Meta::TypeConstraint::Parameterized manpage which should be used.

INHERITANCE

Moose::Meta::TypeConstraint::Parameterizable is a subclass of the Moose::Meta::TypeConstraint manpage.
