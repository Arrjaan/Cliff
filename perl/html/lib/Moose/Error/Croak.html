NAME

Moose::Error::Croak - Prefer croak

VERSION

version 2.0205

SYNOPSIS

    # Metaclass definition must come before Moose is used.
    use metaclass (
        metaclass => 'Moose::Meta::Class',
        error_class => 'Moose::Error::Croak',
    );
    use Moose;
    # ...

DESCRIPTION

This error class uses croak in the Carp manpage to raise errors generated in your metaclass.

METHODS

new

Overrides new in the Moose::Error::Default manpage to prefer croak.
