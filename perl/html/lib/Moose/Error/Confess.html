NAME

Moose::Error::Confess - Prefer confess

VERSION

version 2.0205

SYNOPSIS

    # Metaclass definition must come before Moose is used.
    use metaclass (
        metaclass => 'Moose::Meta::Class',
        error_class => 'Moose::Error::Confess',
    );
    use Moose;
    # ...

DESCRIPTION

This error class uses confess in the Carp manpage to raise errors generated in your metaclass.
