NAME

Moose::Error::Default - the Carp manpage based error generation for Moose.

VERSION

version 2.0205

DESCRIPTION

This class implements the Carp manpage based error generation.

The default behavior is like the Moose::Error::Confess manpage. To override this to default to the Moose::Error::Croak manpage's behaviour on a system wide basis, set the MOOSE_ERROR_STYLE environment variable to croak. The use of this environment variable is considered experimental, and may change in a future release.

METHODS

Moose::Error::Default-new(@args) >>

Create a new error. Delegates to create_error_confess or create_error_croak.

$error-create_error_confess(@args) >>
$error-create_error_croak(@args) >>

Creates a new errors string of the specified style.
