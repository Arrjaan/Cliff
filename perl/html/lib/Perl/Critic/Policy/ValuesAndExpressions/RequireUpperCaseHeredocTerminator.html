NAME

Perl::Critic::Policy::ValuesAndExpressions::RequireUpperCaseHeredocTerminator - Write <<'THE_END'; instead of <<'theEnd'; .

AFFILIATION

This Policy is part of the core Perl::Critic distribution.

DESCRIPTION

For legibility, HEREDOC terminators should be all UPPER CASE letters (and numbers), without any whitespace. Conway also recommends using a standard prefix like "END_" but this policy doesn't enforce that.

  print <<'the End';  #not ok
  Hello World
  the End

  print <<'THE_END';  #ok
  Hello World
  THE_END

CONFIGURATION

This Policy is not configurable except for the standard options.
