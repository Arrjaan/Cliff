NAME

TAP::Parser::Result::Version - TAP syntax version token.

VERSION

Version 3.23

DESCRIPTION

This is a subclass of the TAP::Parser::Result manpage. A token of this class will be returned if a version line is encountered.

 TAP version 13
 ok 1
 not ok 2

The first version of TAP to include an explicit version number is 13.

OVERRIDDEN METHODS
