NAME

File::HomeDir::Driver - Base class for all File::HomeDir drivers

DESCRIPTION

This module is the base class for all the File::HomeDir manpage drivers, and must be inherited from to identify a class as a driver.

It is primarily provided as a convenience for this specific identification purpose, as the File::HomeDir manpage supports the specification of custom drivers and an ->isa check is used during the loading of the driver.
