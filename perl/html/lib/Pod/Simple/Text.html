NAME

Pod::Simple::Text -- format Pod as plaintext

SYNOPSIS

  perl -MPod::Simple::Text -e \
   "exit Pod::Simple::Text->filter(shift)->any_errata_seen" \
   thingy.pod

DESCRIPTION

This class is a formatter that takes Pod and renders it as wrapped plaintext.

Its wrapping is done by the Text::Wrap manpage, so you can change $Text::Wrap::columns as you like.

This is a subclass of the Pod::Simple manpage and inherits all its methods.

SEE ALSO

the Pod::Simple manpage, the Pod::Simple::TextContent manpage, the Pod::Text manpage
