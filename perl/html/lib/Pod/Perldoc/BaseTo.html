NAME

Pod::Perldoc::BaseTo - Base for Pod::Perldoc formatters

SYNOPSIS

    package Pod::Perldoc::ToMyFormat;

    use base qw( Pod::Perldoc::BaseTo );
    ...

DESCRIPTION

This package is meant as a base of Pod::Perldoc formatters, like the Pod::Perldoc::ToText manpage, the Pod::Perldoc::ToMan manpage, etc.

It provides default implementations for the methods

    is_pageable
    write_with_binmode
    output_extension
    _perldoc_elem

The concrete formatter must implement

    new
    parse_from_file

SEE ALSO

the perldoc manpage
