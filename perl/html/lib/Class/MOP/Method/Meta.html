NAME

Class::MOP::Method::Meta - Method Meta Object for meta methods

VERSION

version 2.0205

DESCRIPTION

This is a the Class::MOP::Method manpage subclass which represents meta methods installed into classes by Class::MOP.

METHODS

Class::MOP::Method::Wrapped-wrap($metamethod, %options) >>

This is the constructor. It accepts a the Class::MOP::Method manpage object and a hash of options. The options accepted are identical to the ones accepted by the Class::MOP::Method manpage, except that body cannot be passed (it will be generated automatically).
