NAME

Class::C3::next - Pure-perl next::method and friends

DESCRIPTION

This module is used internally by the Class::C3 manpage when neccesary, and shouldn't be used (or required in distribution dependencies) directly. It defines next::method, next::can, and maybe::next::method in pure perl.
