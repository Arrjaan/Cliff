NAME

nytprofcsv - the Devel::NYTProf::Reader manpage CSV format implementation

SYNOPSIS

 $ nytprofcsv [-h] [-d] [-o <output directory>] [-f <input file>]
 perl -d:NYTProf some_perl_app.pl
 nytprofcsv
 Generating CSV Output...

DESCRIPTION

nytprofcsv is a script that implements the Devel::NYTProf::Reader manpage to create comma-seperated value formatted reports from the Devel::NYTProf manpage databases.

See the the Devel::NYTProf manpage Perl code profiler for more information.

COMMAND-LINE OPTIONS

These are the command line options understood by nytprofcsv

-f, --file <filename>

Specifies the location of the input file. The input file must be the output of fprofpp. Default: nytprof.out

-o, --out <dir>

Where to place the generated report. Default: ./nytprof/

-d, --delete

Purge any existing database located at whatever -o (above) is set to

-h, --help

Print the help message
