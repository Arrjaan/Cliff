Cliff Server

Welcome to the Cliff Server project! Feel free to contribute.

What is Cliff?

Cliff Server bundels Apache, PHP, MySQL, FileZilla Server and many more server deamons in one package! Besides that, Cliff has also a great web adminpanel written in PHP to let you preform server administration in an easy way.
